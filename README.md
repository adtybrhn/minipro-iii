# Instruction
###  :electric_plug: Installation
- Clone this repository.
- Install node modules
```
$ npm install.
```

###  :package: Commands
- To run this project 
```
$ npm start.
```
- To run this fake server 
```
$ npm run server.
```
- To run this fake server & react in parallel 
```
$ npm run dev.
```
